Executive Summary

Key Themes

Internationalisation & Global Reach

Complete internationalisation infrastructure with automated AI-powered translation, translation management UI, and support for all user-facing content.

Accessibility & Alternative Channels

Complete phone-based reporting capability with phone channel support, number management, and voice prompts—expanding access to users without reliable internet or digital literacy.

Cross-Organisational Collaboration

Sibling organisation snapshot sharing with full bidirectional synchronisation of content, files, users, and messages for seamless collaboration between related entities.

Advanced Data Modelling

Repeatable and composite field types for complex data collection, integrated throughout reports tables, analytics, exports, and automation rules.

Case Management Tools

Report linking and grouping to connect related reports and identify patterns, plus enhanced snapshot features for managing complex cases.

Organisation-Wide Task Management

Centralised tasks page with comprehensive filtering, categorisation, time/date scheduling, visual status indicators, and improved task templates.

Analytics & Data Export

Enhanced analytics with XLS export, proper handling of complex fields, toggleable charts, improved filters, and audit log export capabilities.

Multiple security enhancements including CSV injection prevention, EXIF metadata removal, role restrictions, and file viewing audit trails.

Flow editor enhancements with simplified step creation, visual indicators, better validation, and improved error messages for intuitive form creation.

Template-based organisation creation with pre-configured flows, forms, and settings—reducing setup time from hours to minutes.

Automation

Enhanced automation with report attributes in emails, snapshot message triggers, and improved notification delivery for context-aware communications.

Major Features & Improvements

Internationalisation

Multi-Language Translation System

A comprehensive internationalisation infrastructure was implemented to enable the platform to serve organisations worldwide in their native languages.

  • OpenAI Integration: Automated translation using GPT models with batch processing for efficiency
  • Translation Management: Web UI for managing translations with status tracking (translated, outdated, missing)
  • Real-time Updates: WebSocket-based notifications for translation completion
  • Comprehensive Coverage: Translations for flows, surveys, status descriptions, pages, inboxes, and all user-facing content
  • Database-Backed: Translation model with versioning and audit trail
  • Reporter Language Selection: Automatic language detection and manual selection for reporters
  • Build Integration: Frontend language files automatically submitted during deployment
Impact: Enables Elker to serve non-English speaking markets without manual translation work. Organisations can now deploy the platform in any language supported by OpenAI's translation models.
Accessibility

Voice & Phone Reporting System

Complete phone-based reporting capability allowing incident reports to be submitted via telephone, expanding accessibility to users without reliable internet access or digital literacy.

  • Phone Channel Support: New channel type in flow editor for phone-based flows
  • Phone Number Management: Full CRUD operations for organizational phone numbers
  • Calls Log: Dedicated page for tracking all phone interactions
  • Voice Selection: Configurable voice options for automated phone systems
  • Call Settings: Organization-level configuration for phone features
  • Flow Integration: Phone-compatible step types with voice prompts
Impact: Expands platform accessibility to communities with limited digital infrastructure. Particularly valuable for rural areas, elderly populations, and regions with low smartphone adoption.
Collaboration

Sibling Organisation Snapshot Sharing

Organisations can now share snapshots with sibling organisations (related entities within the same parent organisation) with full bidirectional synchronisation.

  • Snapshot Sharing: Share snapshots between sibling organisations with permission controls
  • Content Synchronisation: Automatic sync of snapshot content, fields, and metadata
  • File Synchronisation: Bidirectional file sync - files added or removed in either org reflect in both
  • User Synchronisation: Automatic addition of users to shared snapshot chats
  • Message Sync: Real-time message synchronisation across organisations
  • Report Creation: Create reports from shared snapshots in receiving organisation
  • Activity Tracking: Complete audit trail of snapshot sharing and modifications
  • Notifications: Automatic notifications when snapshots are shared
  • Deletion Sync: Deleting shared snapshot removes it from all sibling organisations
Impact: Enables seamless collaboration between regional offices, departments, or related organisations. Eliminates duplicate data entry and maintains case continuity across organisational boundaries.
Data Management

Repeatable & Composite Report Fields

Advanced field types that support complex data collection scenarios previously requiring workarounds or multiple separate fields.

  • Repeatable Fields: Fields that can be duplicated multiple times (e.g., multiple witnesses, multiple incidents, multiple locations)
  • Composite Fields: Fields containing multiple inputs (e.g., address with street, city, postal code)
  • Reports Table Integration: Repeatable fields display properly in reports table with count indicators
  • Analytics Support: Charts and analytics handle both repeatable and composite field data
  • Export Compatibility: XLS and CSV exports properly format repeatable field data
  • Automation Support: Repeatable fields available in automation rules and email templates
  • Validation: Proper validation for required repeatable fields
  • Flow Editor: Visual editor support for configuring repeatable and composite fields
Impact: Improves data quality by allowing structured capture of complex, multi-instance data. Eliminates workarounds like "Witness 1, Witness 2, Witness 3" fields and provides flexible data modeling.
Task Management

Centralized Tasks Page & Enhanced Task Features

Organization-wide task management with comprehensive filtering, searching, and tracking capabilities.

  • Tasks Page: Dedicated page showing all tasks across all reports with filtering and search
  • Task Categories: Organize tasks into categories for better organization
  • Time & Date: Tasks can now have specific times in addition to dates
  • Status Indicators: Visual indicators for overdue, due today, and upcoming tasks
  • Assignee Filtering: Filter tasks by assignee, status, category, and date range
  • Report Integration: Tasks visible in reports table with count badges
  • Sorting: Sort by due date, priority, assignee, or report
  • Task Templates: Improved task template system for recurring task patterns
Impact: Provides organisation-wide visibility into all pending tasks. Managers can now track all follow-up items across cases in one location, reducing missed deadlines and improving accountability.
Case Management

Report Linking & Grouping

Connect related reports together to identify patterns and track related incidents.

  • Linking Groups: Create groups of related reports with reasons and context
  • Group Reasons: Document why reports are linked (same perpetrator, same incident, related events)
  • Facets & Filters: Filter reports by linked/unlinked status
  • Activity Tracking: History entries when reports are linked or unlinked
  • Group Management: View and manage all report groups from reports table
  • Automatic Grouping: Linking two reports creates a new group (doesn't add to existing)
Impact: Helps identify serial perpetrators, incident patterns, and related cases. Essential for investigations involving multiple reports about the same individual or connected events.
Partner Integration

Partner Dashboard Enhancements

Improved partner integration capabilities for organizations that share data with external partners.

  • Search Functionality: Text search on partner reports list
  • Tasks Index: Partners can view tasks associated with shared reports
  • Task Synchronization: Bidirectional sync between partner tasks and Elker tasks
  • Analytics Support: Partner analytics now support composite and repeatable fields
  • Chart Fixes: Resolved duplicate data in partner analytics charts
  • Status Display: Partners see "Closed" status for completed reports
Impact: Strengthens integration with external partners (law enforcement, regulatory bodies, etc.). Reduces manual coordination and ensures partners have up-to-date information.
Analytics & Reporting

Analytics & Export Improvements

Enhanced data analysis and export capabilities to support organizational reporting requirements.

  • XLS Export: Export analytics and reports to Excel format (in addition to CSV)
  • Composite Field Charts: Analytics charts properly handle composite field data
  • Repeatable Field Support: Charts and tables display repeatable field data accurately
  • Toggleable Charts: Show/hide specific field charts in analytics overview
  • Chart Performance: Fixed duplicate data issues in charts
  • Period Filters: Corrected date range filtering in analytics
  • Audit Log Export: Export audit logs with proper date range filtering
Impact: Provides better data analysis tools and export options. Organisations can now generate comprehensive reports for board meetings, audits, and compliance requirements.
Data Quality

Required Report Attributes & Validation

Improved data quality through mandatory field enforcement and better validation.

  • Required Fields: Report attributes can be marked as required
  • Validation Messages: User-friendly error messages for validation failures
  • Snapshot Designation: Mark which attributes should appear in snapshots
  • System Fields Protection: System report attributes cannot be edited or deleted
  • Field Organization: System fields displayed separately from custom fields
  • Duplicate Prevention: Prevent duplicate slugs and choice names
  • Error Highlighting: Visual highlighting of invalid fields during creation
Impact: Ensures critical data is captured for every report. Reduces incomplete reports and improves data quality for analysis and compliance.
Documentation

Snapshot Functionality Improvements

Enhanced snapshot capabilities for point-in-time data capture and template-based reporting.

  • Snapshot Cloning: Clone snapshots with proper content isolation
  • Same-Org Reports: Create reports from snapshots within the same organization
  • Variables System: Enhanced snapshot variables with composite field support
  • History Tracking: View all snapshot modifications and viewing activity
  • Access Control: Viewer role cannot edit snapshots
  • Source Display: Show snapshot source organization
  • Correspondent Access: Correspondents can create reports from snapshots
  • Token Expiration: Improved SecureMail snapshot token handling
Impact: Streamlines case documentation workflows. Snapshots can serve as templates for similar incidents, reducing data entry time and ensuring consistency.
Audit & Compliance

File Viewing Status Panel

Track and audit document access within the platform.

  • Viewing Tracking: Track which users have viewed each file
  • Timestamp Recording: Record when files were accessed
  • User Identification: Show which specific users viewed files
  • Modal Integration: Viewing status displayed in file preview modal
  • Report & Snapshot Support: Works for both report files and snapshot files
Impact: Provides audit trail for sensitive document access. Important for compliance and understanding information flow within cases.
Privacy

Reporter Anonymity Management

Flexible management of reporter identity and anonymity settings.

  • Anonymity Updates: Managers can change reporter anonymity status from web dashboard
  • Identity Editing: Update reporter contact details and identity information
  • API Support: Full API endpoints across web, reporting, and mobile interfaces
  • History Tracking: Anonymity changes recorded in report history
Impact: Supports evolving privacy needs as cases progress. Allows reporters who initially remained anonymous to later provide identity if they choose.
Authentication

SSO & Authentication Enhancements

Improved single sign-on capabilities and authentication flexibility.

  • Hidden SSO Option: Organisations can hide SSO from login page while keeping it functional
  • Pathway SSO: Added pathway_enabled setting for SSO configurations
  • Direct Link Access: SSO available via direct link even when hidden
  • Configuration UI: Improved SSO settings management interface
Impact: Provides cleaner login experience while maintaining SSO functionality. Useful for organisations with limited SSO adoption or during phased rollouts.
Communication

Messenger & Chat Improvements

Enhanced real-time communication features with better multilingual support.

  • Original Language Display: Show original language text alongside translations
  • Emoji Support: Improved emoji picker across all language settings
  • Messenger Localization: Full translation of messenger interface
  • Group Avatars: Display user avatars in group messages
  • Individual Profiles: Show individual profiles in group chats and case notes
  • Chat Participants: Fixed incorrect display of chat participants
Impact: Improves communication clarity in multilingual environments. Users can see both translated and original text to ensure accurate understanding.

  • Report Details: Modernized report details page layout

  • User Management: Updated user and team management pages
  • Analytics Overview: Redesigned analytics dashboard
  • Audit Log: Improved audit log viewing and filtering
  • Settings Pages: Modernized all settings pages (tags, statuses, roles, levels, report types)

Search & Discovery

Enhanced Search & Filtering

Improved ability to find and filter data across the platform.

  • Dropdown Search: Text search in long dropdown lists (reporter flows)
  • Composite Field Filters: Filter reports by composite field values
  • Period Filters: Fixed date range filtering issues
  • Report Field Filtering: Improved filtering in analytics by report fields
  • Attribute Filters: Enhanced report attribute filtering with better structure
Impact: Reduces time to find specific information. Particularly valuable for large organizations with extensive data and many dropdown options.
Form Building

Flow Editor Enhancements

Improved report form builder with better validation and user experience.

  • Simplified Step Creation: Streamlined dropdown for adding new steps
  • Required Field Indicators: Visual indicators for required fields
  • Choice Validation: Better validation for multiple choice options
  • Auto-fill Support: "Other" option now supported in auto-fills
  • Composite Fields: Visual editor for composite field configuration
  • Phone Channel Filtering: Description step excluded from phone channel flows
  • Error Messages: Improved, user-friendly error messages
  • Template Creation: Fixed flow creation from templates
Impact: Makes form creation more intuitive and reduces configuration errors. Non-technical users can create sophisticated forms without developer assistance.
Administration

Template-Based Organisation Creation

Accelerate new organisation onboarding with pre-configured templates.

  • Organisation Templates: Create organisations from templates with pre-configured settings
  • Flow Copying: Templates include flows and form configurations
  • Field Copying: Report attributes copied from template
  • Settings Inheritance: Organisation settings inherited from template
  • Improved UI: Better organisation creation page with template selection
  • Default Data: Proper handling of default data for new organisations
Impact: Reduces organisation setup time from hours to minutes. Ensures consistency across similar organisations and reduces configuration errors.
Automation

Automation & Notification Enhancements

More powerful and reliable automation capabilities.

  • Report Attributes in Emails: Include report attribute values in automation emails
  • Snapshot Automations: Trigger automations on unread snapshot messages
  • Notification Fixes: Resolved notification delivery issues across various scenarios
  • Reporter Notifications: Fixed notification settings for reporters
  • Assignment Notifications: Improved notifications for snapshot assignments
Impact: Enables more targeted, context-aware automated communications. Reduces manual notification work and ensures stakeholders receive timely updates.
File Management

File & Attachment Improvements

Expanded file support and improved document security.

  • EML File Support: Upload and attach .eml email files
  • Document Previews: Preview doc and excel files in browser
  • EXIF Stripping: Remove EXIF metadata from profile pictures for privacy
  • Preview Fixes: Fixed 500 errors when previewing snapshot files
  • Link Detection: Automatically detect and linkify URLs in text attributes
Impact: Supports more file types commonly used in investigations. EXIF stripping protects user privacy by removing location and device data from photos.
Security

Security Enhancements

Strengthened platform security and access controls.

  • CSV Injection Prevention: Sanitize profile names to prevent CSV injection attacks
  • EXIF Metadata Removal: Strip metadata from images to protect privacy
  • Viewer Restrictions: Viewer role cannot edit reports or snapshots
  • Report Tab Permissions: Fixed permission checks for report tabs
  • Access Control: Improved role-based access enforcement
  • Rollbar Filtering: Fixed overly aggressive data filtering in error logs
Impact: Protects against security vulnerabilities and ensures proper data privacy. Maintains clear access boundaries between user roles.
Performance

Performance Optimisations

Backend optimisations to improve platform speed and scalability.

  • Snapshot Index: Optimised snapshot listing queries
  • Report Fields: Improved report fields index performance
  • Reduced N+1 Queries: Added eager loading to prevent multiple database queries
  • Database Indexes: Added missing indexes and fixed foreign key types
  • Analytics Queries: Optimised analytics data retrieval
Impact: Improves page load times and responsiveness. Supports larger organisations with more data without performance degradation.
